# PINOUT

## LEDs

|     PIN | Note                     |
| ------: | ------------------------ |
| `P0.10` | Red leds in the circle   |
| `P0.09` | Green leds in the circle |
| `P0.22` | Bluetooth status         |
| `P0.21` | Power status             |

## Particulate Matter Sensor

|     PIN | Note   |
| ------: | ------ |
| `P0.05` | PMS Tx |
| `P0.04` | PMS Rx |

## Buttons

|         PIN | Note        |
| ----------: | ----------- |
| SWDIO(nRST) | Side button |
